Download speed

Adaptability and degree of Mobile Friendly

Adaptability - correct display on devices with different screen sizes and resolutions (layout does not float, there is no horizontal scrolling, all content is shown). Mobile Friendly means friendliness to mobile devices (the site is convenient to use on a smartphone or tablet, pictures and text are placed on the screen, pages load quickly).

Usability or user friendliness

Usability - the ability of a site to be user-friendly and understandable. If everything is OK with usability, the user will easily understand the web resource and find what he needs.

Semantic Core is a set of words and phrases by which users can find your site (and, accordingly, by which it will be promoted). You can collect the semantic core manually or using special tools. But it turns out manually is long and difficult, almost no one does that.

Define a list of the main (obvious) words and phrases by which your site will be searched by users. Include logic here, carefully analyze the promoted resource and competitors. As a result, we get the basic keys. Make a table with 2 main columns - phrase and frequency and add keywords to it.

Grouping queries (clustering)

The clustering process is based on determining common landing pages in the TOP of the search engine for a set of keywords and then combining them into groups. Based on these groups, you can already plan the structure of the site and draw up a plan for its filling. To do this, we recommend using specialized tools such as Keyassort, SeRanking, Keycollector.

Website SEO-structure development

The stage is intertwined with the previous one. With competent CL clustering, you will already have an almost ready-made SEO structure: requests are divided into thematic groups, each of which is a set of key phrases for a separate page. Almost, but not 100% finished. Why? Above we wrote about possible branches. And even higher - about 3 clicks. It is necessary to develop a structure in order to "get" into these 3 clicks and not "mess up" with the distribution of semantics.
